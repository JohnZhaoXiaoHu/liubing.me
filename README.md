# liubing.me

基于 VuePress 的个人博客，记录日常开发问题。

## 博客主题

[VuePress Theme Hope](https://theme-hope.vuejs.press/zh/)

## 启动服务

```sh
yarn install # 安装依赖
yarn docs:dev # 启动服务
```
